# 💚 HealthFirst - Patient Health Prediction App

A professional patient management system with blood test tracking and intelligent health risk analysis.

## 🌟 Features

- Add, Edit, Delete, View patient records (Full CRUD)
- Blood test input — Glucose, Haemoglobin, Cholesterol
- Automated health risk analysis from blood values
- Risk badges — Normal / Moderate / High Risk
- Search patients by name or email
- Persistent data storage (no data loss on refresh)
- Professional dashboard with sidebar navigation
- Input validation — email, date of birth, numeric values

## 🛠️ Tech Stack

- React.js — Frontend Framework
- JavaScript — Programming Language
- REST API — Health Analysis Integration
- localStorage — Persistent Storage
- HTML / CSS — Styling

## 🚀 How to Run

git clone healthfirst-app-mihs.vercel.app
npx create-react-app .
cp HealthFirst.jsx src/
npm start

## 📋 Pages

- Dashboard — Patient statistics overview
- Patient Table — All records with search
- Add Patient — Create new record with analysis
- Edit Patient — Update existing record
- Delete Patient — Remove record with confirmation
